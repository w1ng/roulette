# roulette

A Lua implementation of roulette wheel selection using the Alias Method. 

This code was translated from http://www.keithschwarz.com/interesting/code/?dir=alias-method.

# Example
`example.lua`

# 中文说明

轮盘赌选择算法的简单Lua实现,
为确保结果准确,入参的概率总和应该为1,
例如:

`{"a",0.3},{"b",0.3},{"c",0.4}`

`{"a",0.2},{"b",0.1},{"c",0.1},{"d",0.5},{"e",0.1}`
